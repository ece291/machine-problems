Here's a little demo that might help you understand what is going on with the OperandStack in mp2.

Example Input:

3 2 6 4 + / -

First the value 3 gets pushed onto the OperandStack (using SinglePushing).  Remember that you need to have a denominator as well as a numerator.  ***Note, I just made up the starting address of 0000 for the OperandStack -- it probably won't start at address 0000

Address Data  
0000*** 3 ; Numerator <- [StackTop]
0002 1 ; Denomonator
0004    
0006    
0008    
000A    
000C    
000E    
0010    

Then the value 2 gets pushed to the OperandStack.

Address Data  
0000 3 ; Numerator
0002 1 ; Denominator
0004 2 ; Numerator <- [StackTop]
0006 1 ; Denominator
0008 ?  
000A ?  
000C ?  
000E ?  
0010 ?  

This is what the OperandStack will look like after taking in all the values

Address Data  
0000 3 ; Numerator
0002 1 ; Denominator
0004 2 ; Numerator
0006 1 ; Denominator
0008 6 ; Numerator
000A 1 ; Denominator
000C 4 ; Numerator <- [StackTop]
000E 1 ; Denominator
0010 ?  

Then, you hit the first operation,,  "+" in this case.  You need to pop the two most recent values stack..... (and store them somewhere so you can use them later)

Address Data  
0000 3 ; Numerator
0002 1 ; Denominator
0004 2 ; Numerator <- [StackTop]
0006 1 ; Denominator
0008 ?6  
000A ?1  
000C ?4  
000E ?1  
0010 ?  

Now you use the values you got off the stack and do an the addition operation.   The result you should get is (6/1)+(4/1)=10/1.  Then you push the result you get back onto the stack..

Address Data  
0000 3 ; Numerator
0002 1 ; Denominator
0004 2 ; Numerator
0006 1 ; Denominator
0008 10 ; Numerator <- [StackTop]
000A 1 ; Denominator
000C ?4  
000E ?1  
0010 ?  

Now you get do the '/' operation, first getting two values off the stack, doing a calculation and putting the result back onto the stack (2/1) / (10/1)  = (1/5)

Address Data  
0000 3 ; Numerator
0002 1 ; Denominator
0004 1 ; Numerator <- [StackTop]
0006 5 ; Denominator
0008 ?10  
000A ?1  
000C ?4  
000E ?1  
0010 ?  

Finally you hit the '-',,, (3/1) - (1/5) = 14/5

Address Data  
0000 14 ; Numerator <- [StackTop]
0002 5 ; Denominator
0004 ?5  
0006 ?4  
0008 ?10  
000A ?1  
000C ?4  
000E ?1  
0010 ?  

And here we are...  Happy Coding!!!
