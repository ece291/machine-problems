Main Function

This function is really not too different from any other.  It just has a more managerial task as opposed to a specific duty.  It is responsible for the following things:

  • Displaying the menu and only accepting and echoing '1' or '2'.
  • Using DisplayMatrix to display the word grid.
  • Accepting input with ReadWord.
  • If the user had initially chosen KMP, displaying the failure array under the word.
  • Using Dispatch to actually run the correct string matching algorithm.
  • Using DisplayMatrix to display the solution grid.
